## ADR（Architecture Decision Records）

格式：`ADR-<编号> | <标题> | <状态>`

### ADR 模板

```
## ADR-<N>: <标题>

### 状态
提议 / 已采纳 / 已废弃 / 已替代

### 背景
为什么要做这个决策？

### 决策
我们选择了什么？

### 理由
为什么这么选？考虑了哪些替代方案？

### 后果
这个决策带来的好处和代价？
```
